1. **What is the purpose of GitHub?** GitHub is an online version control system that allows users to upload their verson control history for safekeeping outside of a local computer. If the version control hisotry is only stored in git, on a local computer, and that local computer breaks down, the work is lost. By uploading it to GitHub, it can be accessed via other servers. Additionally, GitHub allows users to collaborate on the same repositories through different servers.
